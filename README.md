# Snap2Serve
AI-Based Recipe Recommendation Web Application

## About the Project
Snap2Serve is a web application that recommends recipes based on available ingredients and cooking time. Users upload ingredient images, and AI detects them to generate suitable recipes using Large Language Models (LLMs) and an AI agent. It helps save time, reduce food wastage, and simplify cooking decisions.

## Problem Statement
Users often struggle to decide what to cook with limited ingredients and time. Existing platforms require manual input and lack personalization. This project solves these issues using image-based detection and AI-driven recommendations.

## Technologies Used
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- AI: Image recognition model, LLMs, AI agent  

## Working
1. User uploads ingredient images and enters cooking time  
2. AI detects ingredients  
3. AI agent applies constraints (time, preferences)  
4. LLM generates recipes  
5. Recipes are displayed to the user  

## Features
- Image-based ingredient detection  
- Time-based recipe recommendation  
- AI-powered recipe generation  
- Multi-recipe comparison  
- Cuisine adaptation  
- Step-by-step cooking mode  
- Voice-based instructions  
- Difficulty level indication  
- Dietary filtering  
- Favorites and history  

## Advantages
- Saves time and effort  
- Reduces food wastage  
- Provides personalized suggestions  
- User-friendly interface  

## Stakeholders
- End Users: Use the app to find and cook recipes  
- AI System: Detects ingredients and generates recipes  
- Developers: Build and maintain the application  
- System Administrator: Ensures smooth operation  
- Database: Stores user data and recipes  
- Cloud Provider: Hosts the application  

## System Flow
1. User inputs data via frontend  
2. Backend processes request  
3. Image model detects ingredients  
4. AI agent applies constraints  
5. LLM generates recipes  
6. Results are stored and shown to user  

## Summary
Snap2Serve integrates AI and web technologies to deliver quick, personalized recipe recommendations from available ingredients.
