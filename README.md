# Snap2Serve_mini-project
AI-Based Recipe Recommendation Web Application

# About the Project
This project is a web application that recommends cooking recipes based on ingredients available with the user and the time limit provided.
Users upload images of ingredients, and the system uses AI to detect them and generate suitable recipes using Large Language Models and an AI agent.
The application helps users save time, reduce food wastage, and make quick cooking decisions.<br>

# Problem Statement
Users often find it difficult to decide what to cook with limited ingredients and time.<br>
Existing platforms require manual ingredient input and lack intelligent personalization.<br>
This project addresses these issues using image-based ingredient detection and AI-driven recipe generation.<br>

# Technologies Used
Frontend: React.js<br>
Backend: Node.js, Express.js<br>
Database: MongoDB<br>
AI: Image recognition model, Large Language Models (LLMs), AI agent<br>

# Working 
User uploads ingredient images and enters cooking time.<br>
AI detects ingredients from the images.<br>
An AI agent applies constraints such as time and preferences.<br>
LLM generates suitable recipes.<br>
Recipes are displayed to the user for selection and sharing.<br>

# Features
1.Image-based ingredient detection<br>
2.Time-based recipe recommendation<br>
3.AI-powered recipe generation<br>
4.Multi-recipe comparison<br>
5.Cuisine and regional adaptation<br>
6.Step-by-step cooking mode<br>
7.Voice-based instructions<br>
8.Difficulty level indication<br>
9.Dietary preference and allergy filtering<br>
10.Favorites, history, and recipe sharing<br>
11.AI chatbot assistant<br>

# Advantages
-Saves time and effort<br>
-Reduces food wastage<br>
-Personalized recommendations<br>
-Easy and user-friendly interface


# Stakeholders
- End Users: Use the app to find and cook recipes  
- AI System: Detects ingredients and generates recipes  
- Developers: Build and maintain the application  
- System Administrator: Ensures smooth operation  
- Database: Stores user data and recipes  
- Cloud Provider: Hosts the application
